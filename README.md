# Analizador de Precios de Libros 📚

Script que extrae precios de libros de internet,
los analiza estadísticamente y genera gráficas.

## Tecnologías
- Python
- Pandas
- BeautifulSoup
- Matplotlib

## Qué hace
- Scraping de books.toscrape.com
- Análisis estadístico de precios
- Gráfica de barras ordenada por precio
